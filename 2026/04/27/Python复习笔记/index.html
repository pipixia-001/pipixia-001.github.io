# 函数基础：函数的定义与调用

参数的类型

（1）必须参数（位置参数）:必须严格按照位置参数顺序传参，否则就会报错

def add(name,age):

print(f"我是{name},我今年{age}岁")

a=add("zhongmin",19)

(2)关键字参数：优点在于灵活，可以不用按顺序传参

def ppx(qingshang,zhishang):

print(f"{qingshang}很重要，{zhishang}同样很重要")

b=ppx(zhishang="金钱",qingshang="家人")

'''位置参数和关键字参数同时使用时，关键字参数必须放后边'''

(3)默认参数

'''当不传入新的参数时，使用默认参数，当传入新的参数时，新的参数覆盖原有的默认参数'''

def ming(name,agg=18):

print(f"我的名字是{name},我今年{agg}岁")

c=ming("yangsen")

d=ming("杨森",19)

(4)可变长参数：

'''可变长参数要放在其他参数的后边'''

'''写法一：*后边跟一个单词，默认为args,传入的所有参数会被收集起来形成一个元组,包含位置参数'''

def ppd(*args):

print(args)

ppd(1,3,4,5)

'''写法二：**后边跟一个参数，默认为kwargs,包含关键字参数，收集起来成为一个字典'''

def word(**kwargs):

print(kwargs)

word(name="zhangsan",age=19)

# 高阶函数————实例方法

（1）匿名函数————lambda

'''

用途：一次性使用，简化程序，常用于和map一起使用

格式：lambda 参数：表达式（参数可有可无，但是表达式必须有返回值）

'''

1.无参数的情况

f=lambda:"yangsen"

print(f())

2.有一个参数的情况

f=lambda a:a+2

print(f(5))

print((lambda a:a+2)(5))

3.有多个参数的情况

f=lambda a,b:a**b

print(f(3,2))

print((lambda a,b:a**b)(2,3))

4.lambda的默认参数:默认参数必须放在必须函数的后边

使用默认参数

f=lambda a,b=3:a**b

print(f(3))

print((lambda a,b=4:a**b)(2))

当不使用默认参数时，默认参数被覆盖

print((lambda a,b=3:a+b)(2,4))

(2)高阶函数

1.abs:传入一个数字，返回数字的绝对值

b=abs(-1)

print(b)

print(abs(-3))

2,round（数字，保留位数):保留位数如果省略不写或为0，则默认保留整数位

'''

口诀：四舍六入五去偶

四舍：如果小数部分小于5，则舍掉

六入：如果小数部分大于5，则进一位

五去偶：如果如果小数部分等于5，若前一位是偶数，则舍去；若前一位是奇数，则进位

print(round(3.143,2))#3.14

print(round(4.555,2))#4.56

print(round(3.789,2))#3.79

print(round(4.565,2))#4.56

2.map(func,seq):将函数作用到序列set的每一个元素中，并将结果组成新的迭代器返回

l1=[1,-2,3]

print(list(map(abs,l1)))

f=list(map(lambda x:x+2,l1))

print(f)

filter(func,seq):过滤序列，过滤掉不符合条件的元素，返回一个filter对象:符合lambda条件的将留下，否则，就会被筛选掉

l2=[3,4,5,6,7]

e=list(filter(lambda x:x%2==1,l2))

print(e)

1.any():

'''一个为真即为真'''

score=[45,58,61,89]

a=any(sc>=60 for sc in score)
