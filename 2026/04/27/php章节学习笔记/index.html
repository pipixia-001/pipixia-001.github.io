php数据类型的比较

  • 主要数据类型

    • String（字符串）

    • Boolean（布尔型）

  • 错误抑制符

    • @：在可能出现的表达式前面使用@即可

  • 三目运算符：有三个表达式参与的运算下（简单的分支结构缩写）

    • 语法：表达式?表达式2：表达式3;

  • 连接运算符.

  • 逻辑符

    • 逻辑与：&&
    • 逻辑或:||
    • 逻辑非：!

  • 自操作运算符

    • 分清楚前置自操作和后置自操作

    • ++:在原来基础上加1

    • ——：在原来的基础上减去1

    • +=

    • *=

    • /=

    • %=

  • 计算机码：

    • 原码：数据本身从十进制转换为二进制得到的结果

      • 正数;左边符号为零（正数的原码、反码、补码就是原码本身

      • 负数：左边符号为1

    • 反码：针对负数：符号位不变，其他位取反

    • 补码：针对负数：反码+1

  • 位运算符：

    • &：按位与，两个都为1，结果为1，否则为0

    • |：按位或，两个有一个为1，结果就是1

    • ~：按位非，一个位为1则变成0，否则反之

    • ^:按位异或：两个相同则为零，不同则为1

    • <<: 按位左移，整个位向左移一位，右边补零

    • >>:按位右移，整个位向右移一位，左边补符号位对应内容（正数补0，负数补1）

    • 系统使用任何位运算的时候都是用的补码

# 文件包含

  • 常用的系统函数

    • 有关输出的函数：print() print_r()
    • 有关时间的函数：date() time() microtime()
    • 有关数学的函数：max() min() rand()

  • 文件加载原理

    • 在文件加载的时候，系统会自动将被包含文件中的代码相当于嵌套到当前文件中
    • 加载位置：在哪里加载，对应的文件中的代码嵌入的位置就是对应的include的位置
    • 在php文件中被包含的文件是单独进行编译的

  • Include和require的区别

    • include和require的区别：

      • 本质都是包含文件，唯一的区别是在包含不到文件时，报错形式不一样
      • include的错误级别比较轻，不会阻止代码的执行
      • require的错误级别比较高，如果包含出错，代码将不再执行

    • include和include once的区别：

      • include 系统会碰到一次，执行一次如果对同一个文件进行多次加载，那么系统会执行多次
      • include once：系统碰到多次，也只会执行一次

  • 文件加载的路径：

    • 文件再加载时，需要指定正确的路径才能保证PHP正确的找到对应的文件

    • 文件加载的路径包含两大类：

      • 相对路径：从当前文件所在目录开始的路径：

        1. 绝对路径：从磁盘的根目录开始（本地绝对路径）
        2. 从网站根目录开始（网站绝对路径）

    ### 4. `PHP MySQL `创建数据表

    数据表是数据库的基本存储单元，用于结构化存储数据（比如用户表、文章表）。

    - **核心语法**：`CREATE TABLE 表名 (字段1 数据类型 约束, 字段2 数据类型 约束, ...);`

    - **常用数据类型**：

      表格

      | 类型         | 说明           | 示例                             |
      | :----------- | :------------- | :------------------------------- |
      | `INT`        | 整数           | 存储 ID、年龄                    |
      | `VARCHAR(n)` | 可变长度字符串 | 存储用户名、邮箱（n 为最大长度） |
      | `TEXT`       | 长文本         | 存储文章内容                     |
      | `DATE`       | 日期           | 存储生日、发布时间               |

    - **常用约束**：

      - `PRIMARY KEY`：主键，唯一标识每条记录（通常用 ID 字段）。
      - `NOT NULL`：字段不能为空。
      - `AUTO_INCREMENT`：自动递增，主键常用。
      - `DEFAULT '值'`：设置字段默认值。

      

    - **`PHP` 实现示例（创建用户表）**：
